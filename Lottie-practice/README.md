## Lottie

- Adobe After Effect 로 만든 애니메이션을 다른 플랫폼(Web,AOS,iOS등)에서도 쉽게 사용 할 수 있도록 도와주는 라이브러리 (Airbnb에서 개발)
- 개발자가 직접 애니메이션을 프로그래밍 할 필요 없이, 디자이너가 주는 JSON가지고 재생만 하면됨
